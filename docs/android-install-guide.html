📲 Các bước cài đặt

Bước 1: Bật "Cài đặt từ nguồn không xác định"

Vì ứng dụng chưa đưa lên Google Play Store, bạn cần cho phép cài từ bên ngoài:

  1. Mở Cài đặt (Settings) trên điện thoại
  2. Tìm Bảo mật (Security) hoặc Quyền riêng tư (Privacy)
  3. Bật "Nguồn không xác định" (Unknown sources)
💡 Android 8.0 trở lên: Hệ thống sẽ hỏi riêng cho từng ứng dụng. Khi tải bằng Chrome, chỉ cần nhấn "Cho phép" khi được hỏi.

Bước 2: Tải file APK

  1. Mở trình duyệt Chrome trên điện thoại
  2. Nhấn vào link tải APK phía trên (hoặc quét mã QR nếu có)
  3. Nếu Chrome cảnh báo "Loại file này có thể gây hại...", nhấn "Tải xuống"
  4. Đợi tải xong (khoảng 10-30 giây tuỳ tốc độ mạng)

Bước 3: Cài đặt APK

  1. Khi tải xong, nhấn vào thông báo "Tải xuống hoàn tất" ở thanh trạng thái
    Hoặc mở app Quản lý file → thư mục Downloads
  2. Nhấn vào file adc-delivery-v1.0.0-arm64-v8a.apk
  3. Nhấn "Cài đặt" (Install)
  4. Đợi cài đặt hoàn tất → nhấn "Mở" (Open)

Bước 4: Đăng nhập

  1. Mở ứng dụng ADC Delivery
  2. Nhập email và mật khẩu do quản trị viên cấp
  3. Nhấn Đăng nhập
